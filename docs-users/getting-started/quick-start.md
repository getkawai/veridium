# Quick Start Guide

Get up and running with Kawai DeAI Network in just 5 minutes! This guide will walk you through the essential steps to start using AI services and earning rewards.

## 🚀 5-Minute Setup

### Step 1: Download the App
1. Visit our [releases page](https://github.com/kawai-network/veridium/releases)
2. Download the latest version for your operating system:
   - **macOS**: `Kawai-DeAI-macos.dmg`
   - **Windows**: `Kawai-DeAI-windows.exe`
   - **Linux**: `Kawai-DeAI-linux.AppImage`
3. Install and launch the application

### Step 2: Set Up Your Wallet

#### Option A: Create New Wallet (Recommended for New Users)
1. Click **"Create New Wallet"** in the app
2. **Enter a referral code** (optional):
   - Click "Have a Referral Code?" banner
   - Enter your friend's 6-digit code (e.g., `ABC123`)
   - See bonus upgrade: 5 USDT → **10 USDT** + 100 KAWAI → **200 KAWAI**
3. Generate a new mnemonic phrase (12 or 24 words)
4. **Write down your mnemonic** - keep it safe and private!
5. Create a strong password for your wallet
6. Enter a wallet name (e.g., "My Wallet")
7. Click **"Create Wallet"**

#### Option B: Import Existing Wallet
1. Click **"Import Wallet"**
2. Choose import method:
   - **Mnemonic**: Enter your 12/24 word phrase
   - **Private Key**: Enter your private key
   - **Keystore**: Upload keystore JSON file
3. Enter your password
4. Click **"Import"**

!!! warning "Security First"
    - Never share your mnemonic or private key
    - Store your backup in a secure location
    - Use a strong, unique password
    - Enable 2FA if available

### Step 3: Unlock Wallet & Auto-Claim Trial ✨
1. Enter your wallet password
2. Click **"Unlock Wallet"**
3. **Trial bonus automatically claimed!**
   - Without referral: 5 USDT + 100 KAWAI
   - With referral: **10 USDT + 200 KAWAI**
4. Your balance appears instantly
5. Referral code cleared automatically

!!! tip "Referral Bonus"
    If you entered a referral code in Step 2, you'll automatically receive the upgraded bonus (10 USDT + 200 KAWAI) when you unlock your wallet for the first time. Your friend who referred you will also receive 5 USDT + 100 KAWAI!

### Step 4: Try AI Services
1. Go to the **"AI Chat"** tab
2. Start a conversation with the AI
3. Watch your trial balance decrease as you use services
4. See your cashback rewards accumulate automatically

### Step 5: Explore Rewards
1. Check the **"Rewards"** dashboard
2. View your earning breakdown:
   - Cashback from AI usage
   - Referral bonuses (if applicable)
   - Mining rewards (future)
3. Learn about weekly reward claiming

## 🔧 Troubleshooting

### Common Issues

??? question "Wallet won't connect"
    1. Make sure you have backed up your mnemonic
    2. Add Monad Testnet network manually
    3. Refresh the app and try again
    4. Check our [wallet setup guide](../user-guide/wallet-setup.md)

??? question "Free trial not working"
    1. Ensure wallet is connected
    2. Check if you've already claimed (one per wallet)
    3. Verify you're on Monad Testnet
    4. Contact support if issues persist

??? question "AI services not responding"
    1. Check your USDT balance
    2. Verify internet connection
    3. Try refreshing the app
    4. Report issues on Discord

??? question "Rewards not showing"
    1. Rewards update in real-time for usage
    2. Weekly rewards settle every Monday
    3. Check the correct reward category
    4. Ensure transactions are confirmed

### 🤖 AI Services
- **Chat**: OpenAI-compatible interface with local LLMs
- **Images**: High-quality AI image generation
- **Affordable**: Competitive pricing with cashback

### 💰 Multiple Earning Methods
- **Use-to-Earn**: 5% cashback on every AI request
- **Deposit Cashback**: 1-5% on USDT deposits (tiered)

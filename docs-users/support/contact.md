# Contact Support

Need help? We're here for you! Choose the best way to reach us:

## 🚀 Quick Help

### In-App Support
The fastest way to get help:

1. Open Kawai desktop app
2. Click **Help** in the menu
3. Select **Contact Support**
4. Describe your issue
5. Get response within 24 hours

## 📧 Direct Contact

### Email Support
For detailed issues or private matters:

**support@kawai.network**

Response time: 24-48 hours

Please include:
- Your wallet address (first 6 and last 4 characters)
- Description of the issue
- Screenshots if applicable
- Steps to reproduce (for bugs)

### Business Inquiries
For partnerships and business:

**business@kawai.network**

## 🐛 Report Bugs

### GitHub Issues

## 📋 Before You Contact

Help us help you faster! Please have ready:

### For Technical Issues
- [ ] Wallet address (public)
- [ ] Operating system and version
- [ ] App version (Help → About)
- [ ] Error message or screenshot
- [ ] Steps to reproduce

### For Reward Issues
- [ ] Wallet address
- [ ] Transaction hash (if applicable)
- [ ] Settlement period
- [ ] Expected vs actual amount

### For Account Issues
- [ ] Wallet address
- [ ] Last successful login date
- [ ] Description of the problem

- Check domain names carefully
- Look for verified badges

# Welcome to Kawai DeAI Network

**Decentralized AI Compute Network on Monad Blockchain**

Kawai DeAI Network is a revolutionary platform that brings AI computing power to everyone while rewarding users for participation. Whether you're using AI services or contributing compute power, you earn KAWAI tokens through our innovative reward systems.

## 🚀 What is Kawai DeAI?

Kawai DeAI Network combines the power of decentralized computing with AI services, creating a sustainable ecosystem where:

- **Users** get affordable AI chat and image generation services
- **Contributors** earn tokens by providing GPU compute power  
- **Everyone** benefits from a fair, transparent reward system

## ✨ Key Features

### 🤖 AI Services
- **Chat with AI**: OpenAI-compatible chat interface with local LLM models
- **Image Generation**: High-quality AI image creation using Google Gemini API
- **Affordable Pricing**: Pay with USDT, earn cashback in KAWAI tokens

### 💰 Multiple Ways to Earn
- **Use-to-Earn**: Get 5% cashback on every AI request
- **Deposit Cashback**: Earn 1-5% KAWAI tokens on USDT deposits (tiered system)
- **Referral Rewards**: Earn 5% commission from friends' mining rewards
- **Mining Rewards**: Contribute GPU power and earn up to 90% of generated tokens
- **Hold-to-Earn**: KAWAI holders receive 100% of platform revenue in USDT

### 🔄 P2P Trading
- **Decentralized Marketplace**: Trade KAWAI ↔ USDT directly with other users
- **No Middleman**: Peer-to-peer trading with smart contract security
- **Fair Pricing**: Market-driven price discovery

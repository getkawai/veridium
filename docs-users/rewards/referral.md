## 🆘 Troubleshooting

### "My referral code isn't working"

**Check:**
- Is it exactly 6 characters?
- All uppercase (ABC123 not abc123)?
- No spaces before/after?
- Code was actually generated?

**Solutions:**
- Regenerate code if needed
- Copy using the copy button
- Test code yourself (different device)

### "Friend signed up but I didn't get reward"

**Possible causes:**
1. They didn't enter code during signup
2. They used different code
3. Anti-abuse check flagged (same device/IP)
4. System delay (wait 1 minute)

**Verify:**
- Check your referral dashboard
- Ask friend to check their profile ("Referred by")
- Wait 5 minutes and refresh

### "Commission not showing"

**Check:**
- Has your referral actually mined?
- Is it after weekly settlement (Monday)?
- Check dashboard: Wallet → Rewards → Referral → Commission tab

**Timeline:**
```
Referral mines: Tuesday
Your commission recorded: Instantly
Settlement: Next Monday
Claimable: Monday onwards
```

### "Can't claim my commission"

**Requirements:**
- Must be after Monday settlement
- Must have MON for gas
- Wallet must be connected
- Sufficient commission amount

**Solutions:**
- Wait for settlement
- Get MON from faucet
- Reconnect wallet
- Check explorer for contract status

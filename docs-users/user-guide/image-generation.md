## 💎 Quality Tiers

### Standard (1K)

- **Resolution**: 1024px
- **File size**: 200-500 KB
- **Speed**: Fast (~10 sec)
- **Cost**: ~0.05 USDT
- **Best for**: Social media, web use

### HD (2K)

- **Resolution**: 2048px
- **File size**: 500KB-1MB
- **Speed**: Medium (~20 sec)
- **Cost**: ~0.10 USDT
- **Best for**: High-quality web, small prints

### Ultra (4K)

- **Resolution**: 4096px
- **File size**: 1-3 MB
- **Speed**: Slower (~30 sec)
- **Cost**: ~0.15-0.20 USDT
- **Best for**: Large prints, professional use

!!! tip "Model Selection Priority"
    - **Explicit model selection** > quality setting > default
    - Selecting "Gemini Pro" + "Standard" will use Pro model at 1K
    - Selecting "Auto" + "Ultra" will use Pro model (required for 4K)

## ✍️ Writing Great Prompts

### Basic Structure

**Format:**
```
[Subject] + [Style] + [Details] + [Mood/Lighting]
```

**Example:**
```
A majestic lion, digital art, golden mane flowing in wind, sunset lighting
```

### Prompt Tips

**1. Be specific**
```
❌ "A car"
✅ "A red sports car, Ferrari 458, racing on mountain road, motion blur"
```

**2. Add style keywords**
```
- "digital art"
- "oil painting"
- "photorealistic"
- "anime style"
- "watercolor"
- "3D render"
```

**3. Include details**
```
✅ "Cherry blossom tree, spring season, pink petals falling,
    Japanese garden background, peaceful atmosphere"
```

**4. Specify lighting**
```
- "golden hour lighting"
- "dramatic shadows"
- "soft ambient light"
- "neon lights"
- "studio lighting"
```

**5. Set the mood**
```
- "peaceful and serene"
- "epic and dramatic"
- "dark and mysterious"
- "bright and cheerful"
```

### Prompt Examples

**Landscape:**
```
Mountain landscape at sunrise, snow-capped peaks, misty valleys,
golden light, photorealistic, 8K quality
```

**Portrait:**
```
Portrait of a young woman, elegant dress, soft studio lighting,
professional photography, shallow depth of field
```

**Fantasy:**
```
Dragon flying over medieval castle, fantasy art style, dramatic clouds,
ray of sunlight, epic composition
```

**Sci-Fi:**
```
Futuristic city at night, neon lights, flying cars, cyberpunk style,
rain reflections, blade runner aesthetic
```

**Abstract:**
```
Abstract geometric shapes, vibrant colors, flowing patterns,
modern art, digital illustration
```

## 🆘 Troubleshooting

### "Generation Failed"

**Common causes:**
1. Prompt violates content policy
2. Network timeout
3. Server overload

**Solutions:**
1. Rephrase your prompt
2. Try again in a few seconds
3. Switch to different model

### Poor Quality Results

**Solutions:**
1. **Be more specific** in prompt
2. **Try different model** (Pro vs Flash)
3. **Increase quality** setting
4. **Add style keywords**
5. **Reference similar images** you like

### Wrong Aspect Ratio

**Solution:**
- Double-check aspect ratio selection
- Some platforms crop images (account for this)
- Generate at higher resolution, then crop

### Image Artifacts

**If you see glitches or distortions:**
1. **Regenerate** (might be random issue)
2. **Simplify prompt** (too complex)
3. **Try Pro model** (better quality)

### "Insufficient Balance"

**Solution:**
1. Check USDT balance
2. [Deposit more USDT](deposit.md)
3. Or adjust quality (use Standard)

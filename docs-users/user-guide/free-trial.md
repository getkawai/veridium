## 🔐 Anti-Abuse Protection

### Why Limits Exist

To ensure fair distribution of free trials:

- ✅ One trial per wallet address
- ✅ One trial per device (Machine ID)
- ✅ Prevents multiple claims by same person
- ✅ Keeps the system sustainable

### What This Means

**You can:**
- ✅ Use trial on your device
- ✅ Refer unlimited friends (they get separate trials)
- ✅ Deposit more USDT anytime

**You can't:**
- ❌ Claim trial twice on same wallet
- ❌ Claim trial twice on same device
- ❌ Use your own referral code (self-referral)

!!! warning "One Device, One Trial"
    If you reinstall the app or create a new wallet on the same device, you won't get another trial. The device fingerprint prevents this.

## 🆘 Troubleshooting

### "Free Trial Already Claimed"

**Reasons:**
1. You used this wallet before
2. You used this device before
3. Trial was already claimed

**Solutions:**
- Use a different wallet address
- Use a different device
- [Deposit USDT](deposit.md) to continue

### "Referral Code Invalid"

**Check:**
- Is it 6 characters exactly?
- All uppercase (ABC123 not abc123)
- No special characters
- No spaces

**Still not working?**
- Ask friend to share code again
- Try copying and pasting
- Contact support if persistent

### Balance Not Showing

**Solutions:**
1. **Refresh the page**
2. **Restart the app**
3. **Check network** (should be Monad)
4. **Wait 30 seconds** (blockchain confirmation)

### Trial Claimed But No Balance

**Verify:**
1. Check wallet address is correct
2. Check network is Monad Testnet
3. Check trial status in Settings
4. Check transaction on [explorer](https://testnet.monad.xyz)

**If problem persists:**
- [Contact support](../support/contact.md)
- Include wallet address and timestamp

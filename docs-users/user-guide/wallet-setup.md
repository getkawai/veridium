## 🚀 Creating Your Wallet

### Step 1: Welcome Screen

When you first open the app:

1. You'll see: **"Welcome To Kawai DeAI Network"**
2. Bonus display: "Get 5 USDT + 100 KAWAI FREE"
3. Click **"Setup wallet & Claim Bonus"**

### Step 2: Apply Referral Code (Optional)

!!! tip "Double Your Bonus!"
    If you have a referral code, enter it NOW to get 10 USDT + 200 KAWAI instead of 5 + 100!

**If you see "Have a Referral Code?" banner:**

1. Click **"Enter Code"** to expand
2. Type your **6-character code** (e.g., `ABC123`)
3. Click **"Apply"**
4. Banner shows: **"🎉 Bonus Upgraded!"**
5. New bonus: **10 USDT + 200 KAWAI**

**Don't have a code?** Skip this step - you still get 5 USDT + 100 KAWAI!

### Step 3: Create Password

**Create a strong password to encrypt your wallet:**

1. Enter password (minimum 8 characters)
2. Password strength indicator shows:
   - **Weak** (red) - Add more characters
   - **Fair** (orange) - Add uppercase or numbers
   - **Good** (green) - Add special characters
   - **Strong** (blue) - Perfect!

3. Confirm password (enter same password again)

!!! warning "Password Security"
    This password encrypts your wallet on your device. Choose a strong, unique password!

**Password tips:**
- ✅ At least 12 characters
- ✅ Mix uppercase and lowercase
- ✅ Include numbers
- ✅ Include special characters (!@#$%^&*)
- ❌ Don't use common words
- ❌ Don't reuse passwords from other sites

### Step 4: Choose Wallet Method

You have **4 options**:

#### Option 1: Generate New Mnemonic (Recommended for New Users)

**Best if:** You're new to crypto or want a fresh wallet.

1. Click **"Generate New Mnemonic"**
2. App creates a 12-word phrase for you
3. Continue to backup step →

#### Option 2: Use Existing Mnemonic (Import from Another Wallet)

**Best if:** You already have a wallet (MetaMask, Trust Wallet, etc.)

1. Click **"Use Existing Mnemonic"**
2. Enter your 12 or 24-word phrase
3. Enter wallet name (e.g., "My Main Wallet")
4. Click **"Import Wallet"**

#### Option 3: Import Keystore (From JSON File)

**Best if:** You exported keystore from MetaMask or similar.

1. Click **"Import Keystore"**
2. Click **"Select Keystore File"** or paste JSON
3. Enter the keystore password
4. Optional: Add wallet description
5. Click **"Import Keystore"**

#### Option 4: Import Private Key (Advanced)

**Best if:** You have a raw private key (64 hex characters).

1. Click **"Import Private Key"**
2. Enter your private key (with or without `0x` prefix)
3. Format validation shows: "✓ Valid format" or character count
4. Create password for this wallet
5. Enter wallet name
6. Click **"Import Private Key"**

!!! danger "Private Key Security"
    Never share your private key! Anyone with it can steal your funds.

### Step 5: Backup Your Mnemonic (New Wallet Only)

**If you generated a new mnemonic, this is CRITICAL:**

1. You'll see **12 words** displayed clearly
2. Click the **copy button** (top right) or write them down
3. Store securely (see security tips below)
4. Enter wallet name (e.g., "My Main Wallet")
5. Click **"I have written it down"**

!!! danger "Critical - Read This!"
    Your 12-word mnemonic is the ONLY way to recover your wallet. If you lose it AND forget your password, your funds are GONE FOREVER!

**Backup checklist:**
- [ ] Written on paper (not digital)
- [ ] Stored in secure location
- [ ] Made 2-3 copies in different places
- [ ] Never shared with anyone
- [ ] Never stored in cloud/email/screenshots

### Step 6: Wallet Created!

**Success! Your wallet is ready.**

- ✅ Bonus automatically claimed (5 or 10 USDT)
- ✅ Wallet encrypted on your device
- ✅ Ready to use AI services
- ✅ Can deposit more USDT anytime

**Backup reminder modal will appear** - make sure you've backed up your mnemonic!

## 🔄 Managing Multiple Wallets

You can create/import multiple wallets in the app.

### Adding More Wallets

1. Click **"Unlock"** screen
2. Click **"Add Wallet"** button
3. Follow the same setup process
4. Choose any import method

### Switching Between Wallets

1. From unlock screen, select wallet from dropdown
2. Enter password
3. Click **"Unlock"**
4. Active wallet switches automatically

### Managing Existing Wallets

1. Click **"Manage Wallets"** from unlock screen
2. See all your wallets:
   - Wallet name and address
   - Active status (green dot)
   - Export and delete options

**Actions available:**
- **Export Keystore** - Download JSON backup (📥 icon)
- **Delete Wallet** - Remove from app (🗑️ icon)
  - Only non-active wallets can be deleted
  - Confirm before deletion (irreversible!)

!!! warning "Deleting Wallets"
    Make sure you have backup (mnemonic or keystore) before deleting a wallet!

## 🔐 Security Best Practices

### Protect Your Mnemonic Phrase

!!! danger "Critical - Read Carefully!"
    Your 12-word mnemonic is the ONLY way to recover your wallet if you lose/forget your password. If you lose BOTH password AND mnemonic, your funds are GONE FOREVER!

**The Two-Layer Security:**
1. **Password** - Unlocks wallet on your device
2. **Mnemonic** - Recovers wallet if you lose password/device

**Backup your mnemonic immediately after creation!**

**Do:**
- ✅ Write on paper (2-3 copies)
- ✅ Store in safe/secure location
- ✅ Keep separate from password
- ✅ Test recovery on another device (optional but recommended)

**Don't:**
- ❌ Store in cloud (Google Drive, Dropbox)
- ❌ Take screenshots or photos
- ❌ Share with anyone (even "support")
- ❌ Store in email or notes apps
- ❌ Type into untrusted websites

### Password Best Practices

**Your password protects local access:**

- ✅ Use strong, unique password
- ✅ Different from mnemonic
- ✅ Store in password manager (optional)
- ✅ Never share

**If you forget password:**
- Import wallet again using mnemonic
- Create new password
- All funds will be accessible

### Transaction Safety

**The app will ask for password when:**
- Depositing USDT
- Claiming rewards
- Trading on marketplace
- Any blockchain transaction

**Before confirming:**
1. **Verify amount** - Is it what you intended?
2. **Check recipient** - Is it the correct contract?
3. **Review action** - Understand what's happening
4. **Gas fee** - Should be ~0.001 MON (very low)

### Scam Prevention

**Official Kawai will NEVER:**
- ❌ Ask for your mnemonic phrase
- ❌ Ask for your private key
- ❌ Ask for your password
- ❌ DM you first on Discord/Telegram
- ❌ Ask you to send tokens for "verification"

**Common scams:**
1. **Fake support** - "Help" that asks for private info
2. **Phishing** - Fake Kawai apps or websites
3. **Too good to be true** - "Guaranteed" returns scams
4. **Urgency tactics** - "Act now or lose access!"

**If suspicious:** Ask in official Discord or check documentation.

## 🆘 Troubleshooting

### "Invalid Password" When Unlocking

**Possible causes:**
1. Wrong password
2. Wrong wallet selected (if multiple)
3. Caps Lock on

**Solutions:**
- Try password again (check Caps Lock)
- Select correct wallet from dropdown
- If truly forgotten: Import wallet using mnemonic

### "Invalid Mnemonic" During Import

**Common issues:**
1. Wrong number of words (must be 12 or 24)
2. Typos or extra spaces
3. Invalid words (not in BIP39 wordlist)
4. Wrong order

**Solutions:**
- Count words (should be exactly 12 or 24)
- Remove extra spaces between words
- Check for typos
- Verify word order (sequence matters!)
- All words should be lowercase

### "Invalid Private Key Format"

**Requirements:**
- Exactly 64 hexadecimal characters
- Only 0-9 and a-f (case insensitive)
- With or without `0x` prefix

**Example valid format:**
```
0x1234567890abcdef1234567890abcdef1234567890abcdef1234567890abcdef
```

### Can't Import Keystore

**Check:**
- Is the JSON file valid? (try opening in text editor)
- Is the password correct? (the one used to export keystore)
- File not corrupted?

**Solution:**
- Re-export keystore from original wallet
- Try copying JSON content and pasting instead of file upload

### Wallet Not Showing After Creation

**This should not happen, but if it does:**
1. Restart the app
2. Try unlocking with your password
3. Check "Manage Wallets" to see if it's listed
4. If lost: Import using your mnemonic backup

### Forgot Password

**Don't panic! Your funds are safe.**

**Recovery process:**
1. You have your mnemonic backed up, right? ✅
2. Delete the wallet (or wait for "Import wallet" option)
3. Import using your mnemonic phrase
4. Create NEW password
5. Wallet restored with all funds!

**If you lost BOTH password AND mnemonic:**
- Unfortunately, there's no way to recover 😔
- This is why backup is critical!

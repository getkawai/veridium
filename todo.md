# Pencarian Video di Komputer

## Tujuan
Mencari semua file video yang tersimpan di komputer

## Langkah-langkah
- [ ] Mencari file video dengan ekstensi umum (.mp4, .avi, .mkv, .mov, .wmv, .flv, .webm, .m4v)
- [ ] Mencari file video dengan ekstensi menos (.3gp, .ogv, .ts, .mts, .vob, .asf, .rm, .rmvb)
- [ ] Mengorganisir hasil pencarian
- [ ] Menyajikan daftar video yang ditemukan
